# SistemaServiciosTecMD - Cálculo de Servicios Públicos

Es una aplicación de escritorio programada en Java para calcular el cobro de facturas de agua, energía y gas. Fue desarrollado para la materia de **Análisis y Diseño de Sistemas** en el programa de **Técnico en Servicios de Seguridad Informática**.

## Características del proyecto
* **Uso de programación orientada a objetos**: Aplica conceptos de herencia y polimorfismo para liquidar tarifas personalizadas según el estrato socioeconómico.
* **Interfaz de escritorio**: Diseñado con Java Swing para permitir el ingreso de datos de consumo y visualizar los montos netos a pagar.
* **Modelado del sistema**: Incluye diagramas UML detallados sobre el funcionamiento y diseño de las clases.

## Archivos del repositorio
* `src/`: Código fuente en Java estructurado con la interfaz y los modelos de cálculo.
* `docs/`: Documento final en PDF que contiene los diagramas de casos de uso, clases, objetos, secuencias y componentes.
